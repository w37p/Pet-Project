Структура UX (Каркас логики)


Пользовательский путь:



1. Главный экран
   
   → Выбор адреса
   
   → Категории доставок:
   
      - Еда
        
      - Напитки
        
      - Маркет / Аптека
        
      - Всё рядом




3. Карточки категорий



   → Меню товаров:
   
      - Фото блюда
  
      - Название
        
      - Цена

      - Кнопка "Добавить"



5. Карточка товара:


   
      - Фото блюда
        
      - Название
        
      - Описание
        
      - Цена
        
      - Кнопка "Добавить"
  
        

7. Корзина


   
   → Итог заказа:
   
      - Товары
        
      - Стоимость

      - Кнопка "Оформить"



9. Профиль



   → История заказов
   
   → Помощь / поддержка
